# Weather Checker
--- 

## What it does:
---
- Uses an api (Open-Meteo) to fetch weather data.
- Reads weather data 
- Returns information about location, weather, and reccommendations.

## How to run it:
---
- Open file within a text editor or navigate to file using terminal.
- Run in the text editor or bash python3 main.py if using terminal.
